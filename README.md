# Unity-Simple-FPS-2025
## Made in version 2022.1.23f1

1. Clone this repo to your computer OR just download the project files
2. Open Unity HUB and click the arrow next to "Open" button and then "Add project from disk"
3. Choose the project folder (when you have cloned the repo, choose the project folder inside the repo folder)
4. Now it should open the project
